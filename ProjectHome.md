# Download HE ROBOT #

## Requirements ##

  * Python 2.4

  * Xwindow or similar

## Running He Robot ##

1. Download [herobot-10.zip](http://herobot.googlecode.com/files/herobot-10.zip)

2. Unzip the file:
```
$ unzip herobot-10.zip
```

3. Run the game
```
$ cd herobot ; python run_game.py
```
