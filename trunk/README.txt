He Robot
========

Entry in PyWeek #5  <http://www.pyweek.org/5/>
Team: Palangana and Tourus
Members: flecox, luis, hikaru, noe, nueces, r0ver, emanuel, frapell, quimera

DEPENDENCIES:

You might need to install some of these before running the game:

  Python:     http://www.python.org/
  PyGame:     http://www.pygame.org/

RUNNING THE GAME:

On Windows or Mac OS X, locate the "run_game.pyw" file and double-click it.
Othewise open a terminal / console and "cd" to the game directory and run:

  python run_game.py

HOW TO PLAY THE GAME:
Using the mouse move the *crosshear* and shot with a click. In the game you have to kill as much humans as you can. Don't kill robots.
